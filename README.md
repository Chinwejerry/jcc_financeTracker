Finance Tracker App 💰

A lightweight web application for tracking personal finances, including income, expenses, and savings. This app helps users visualize their spending habits and maintain better financial control.

Features
Add and manage income and expenses
View transaction history
Categorize spending (e.g., food, rent, entertainment)
Basic dashboard with summaries
Simple, responsive UI for desktop and mobile
Getting Started
Clone the repository
Install dependencies
Run the development server
Open the app in your browser
npm install
npm run dev
Configuration

Create a .env file in the root directory to store environment variables such as API URLs and database credentials.

API_URL=your_api_url_here
DATABASE_URL=your_database_url_here
Authentication

To access the app, users must log in with valid credentials.

Login Instructions
Go to the login page: /login
Enter your registered email and password
Click Sign In to access your dashboard

New users must register before logging in.

Tech Stack
Frontend: React 
Backend: Node.js / Express
Database: PostgreSQL / MongoDB
